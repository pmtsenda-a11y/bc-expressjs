# 📚 Libros Gratuitos — Semana 13: OpenAPI / Swagger

## Recursos disponibles en línea

### 1. OpenAPI Specification (Documentación Oficial)
- **URL**: https://spec.openapis.org/oas/latest.html
- **Descripción**: Especificación completa de OpenAPI 3.1.0 mantenida por la
  OpenAPI Initiative. Referencia autoritativa para todos los campos, tipos y
  objetos de la spec.
- **Capítulos relevantes**: Information Object, Paths Object, Components Object,
  Security Scheme Object

### 2. Swagger Documentation Hub
- **URL**: https://swagger.io/docs/
- **Descripción**: Colección de guías, tutoriales y referencia de la spec
  publicada por SmartBear. Incluye ejemplos prácticos y comparativas.
- **Secciones clave**:
  - [Getting Started](https://swagger.io/docs/open-source-tools/swagger-ui/usage/installation/)
  - [OpenAPI Guide](https://swagger.io/docs/specification/about/)

### 3. Petstore OpenAPI Example (referencia de spec completa)
- **URL**: https://petstore.swagger.io/
- **Descripción**: La demo oficial de Swagger UI con una spec completa. Útil para
  explorar cómo se documentan distintos tipos de endpoints, schemas y seguridad.
- **Spec JSON**: https://petstore.swagger.io/v2/swagger.json

### 4. OpenAPI Map (diagrama interactivo)
- **URL**: https://openapi-map.apihandyman.io/
- **Descripción**: Mapa visual e interactivo de todos los objetos de la
  especificación OpenAPI 3.x. Permite explorar la jerarquía completa de la spec
  haciendo clic en cada nodo.
