# Ebooks Gratuitos — Semana 08

## Seguridad en Node.js y APIs REST

### OWASP Testing Guide v4.2
- **URL**: https://owasp.org/www-project-web-security-testing-guide/
- **Descripción**: Guía oficial de OWASP para testing de seguridad web. Capítulos relevantes: OTG-AUTHN (autenticación), OTG-AUTHZ (autorización), OTG-INPVAL (validación de inputs).
- **Formato**: PDF / HTML online

### Node.js Security Handbook — Snyk
- **URL**: https://go.snyk.io/rs/677-THP-415/images/Node.js_security_handbook.pdf
- **Descripción**: Handbook gratuito de Snyk sobre seguridad en Node.js. Cubre inyecciones, autenticación, dependencias vulnerables y configuración segura.
- **Formato**: PDF gratuito (requiere email)

### OWASP Cheat Sheet Series
- **URL**: https://cheatsheetseries.owasp.org/
- **Descripción**: Colección de guías concisas sobre temas de seguridad específicos. Especialmente útiles para esta semana:
  - [Access Control Cheat Sheet](https://cheatsheetseries.owasp.org/cheatsheets/Access_Control_Cheat_Sheet.html)
  - [Authentication Cheat Sheet](https://cheatsheetseries.owasp.org/cheatsheets/Authentication_Cheat_Sheet.html)
  - [Input Validation Cheat Sheet](https://cheatsheetseries.owasp.org/cheatsheets/Input_Validation_Cheat_Sheet.html)
- **Formato**: HTML online / PDF descargable

### Express.js Security Best Practices (Documentación Oficial)
- **URL**: https://expressjs.com/en/advanced/best-practice-security.html
- **Descripción**: Guía oficial de Express sobre mejores prácticas de seguridad en producción. Cubre Helmet, rate limiting, HTTPS y más.
- **Formato**: HTML online (gratuito)
