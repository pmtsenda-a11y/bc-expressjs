# 📚 eBooks y Documentación Gratuita

## Docker Official Documentation

- **Título**: Docker Docs — Get Started
- **URL**: https://docs.docker.com/get-started/
- **Descripción**: Tutorial oficial de Docker desde cero. Cubre Dockerfile,
  contenedores, imágenes, docker-compose y publicación en Docker Hub.

---

## Play with Docker

- **Título**: Play with Docker Classroom
- **URL**: https://training.play-with-docker.com/
- **Descripción**: Laboratorios interactivos gratuitos en el navegador.
  Sin necesidad de instalar Docker localmente. Ideal para practicar comandos
  y flujos reales.

---

## Docker Curriculum

- **Título**: Docker Curriculum (prakhar.me)
- **URL**: https://docker-curriculum.com/
- **Descripción**: Guía completa gratuita que cubre desde los fundamentos hasta
  el despliegue en AWS. Escrita en inglés, con ejemplos prácticos y progresivos.

## Dockerfile Best Practices (Docker)

- **Título**: Best practices for writing Dockerfiles
- **URL**: https://docs.docker.com/develop/develop-images/dockerfile_best-practices/
- **Descripción**: Guía oficial con patrones recomendados para reducir el tamaño
  de imágenes, aprovechar el caché de capas y mejorar la seguridad.
