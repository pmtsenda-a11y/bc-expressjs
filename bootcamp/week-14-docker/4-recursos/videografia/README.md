# 🎬 Videografía Recomendada

## Docker in 100 Seconds — Fireship

- **Canal**: Fireship
- **URL**: https://www.youtube.com/watch?v=Gjnup-PuquQ
- **Duración**: ~2 min
- **Descripción**: Introducción visual y concisa a los conceptos clave de Docker.
  Ideal como primer contacto antes de la teoría.

## Docker Tutorial for Beginners — TechWorld with Nana

- **Canal**: TechWorld with Nana
- **URL**: https://www.youtube.com/watch?v=3c-iBn73dDE
- **Duración**: ~3 horas
- **Descripción**: Tutorial completo que cubre Docker, docker-compose, redes,
  volúmenes y el flujo de trabajo profesional con contenedores.

## Docker Crash Course — Traversy Media

- **Canal**: Traversy Media
- **URL**: https://www.youtube.com/watch?v=fqMOX6JJhGo
- **Duración**: ~2 horas

## Multi-stage Docker Builds — Theo (t3.gg)

- **Canal**: Theo - t3.gg
- **URL**: https://www.youtube.com/watch?v=SnSH8Ht3MIc
- **Duración**: ~20 min
