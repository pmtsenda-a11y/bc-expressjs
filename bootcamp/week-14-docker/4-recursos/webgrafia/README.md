# 🌐 Webgrafía y Documentación

## Documentación Oficial de Docker

- **URL**: https://docs.docker.com/
- **Descripción**: Referencia completa de Docker: Dockerfile, CLI, docker-compose,
  redes, volúmenes y despliegue.

## Docker Hub — Imagen Oficial de Node.js

- **URL**: https://hub.docker.com/_/node
- **Descripción**: Página oficial de la imagen `node` en Docker Hub. Muestra las
  variantes disponibles (`slim`, `alpine`, `bookworm`), sus tamaños y qué
  casos de uso resuelve cada una.

## Docker Compose File Reference

- **URL**: https://docs.docker.com/reference/compose-file/
- **Descripción**: Referencia completa de todas las directivas de `docker-compose.yml`:
  `services`, `volumes`, `networks`, `depends_on`, `healthcheck` y más.
